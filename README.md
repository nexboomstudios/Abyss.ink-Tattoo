# ABYSS.INK

A premium brutalist tattoo studio website concept built by NexBoom Studios.  
NexBoom Studios tarafından geliştirilen premium brutalist tattoo studio website konsepti.

---

## Overview / Proje Hakkında

**ABYSS.INK** is a dark, premium, and immersive tattoo studio web experience focused on trust, craftsmanship, identity, and visual storytelling.

The project explores how editorial aesthetics, smooth interactions, premium motion, and brutalist visual language can create a memorable digital experience for a modern tattoo brand.

**ABYSS.INK**, güven, sanat anlayışı, kimlik ve görsel hikâye anlatımı odağında tasarlanmış karanlık ve premium bir tattoo studio deneyimidir.

Bu proje; editorial estetik, akıcı etkileşimler, premium motion tasarımı ve brutalist görsel dilin modern bir tattoo markası için güçlü bir dijital deneyime nasıl dönüşebileceğini keşfeder.

## Design Philosophy / Tasarım Felsefesi

ABYSS.INK is designed to balance **trust and darkness**.

Rather than relying on cliché tattoo aesthetics, the project focuses on luxury editorial visuals, controlled motion, brutalist typography, and premium spacing to create a studio experience that feels professional, artistic, and memorable.

ABYSS.INK; **güven ve karanlık atmosfer** arasında denge kurmak amacıyla tasarlanmıştır.

Klişe tattoo estetiğine yaslanmak yerine; premium editorial görseller, kontrollü motion tasarımı, brutalist tipografi ve güçlü boşluk kullanımıyla profesyonel, sanatsal ve akılda kalıcı bir studio deneyimi yaratmayı hedefler.

## Built Without / Kullanılmayan Teknolojiler

This project intentionally avoids heavy frameworks and unnecessary dependencies to preserve performance and maintain full creative control.

Bu proje; performansı korumak ve tam tasarım kontrolü sağlamak amacıyla ağır framework ve gereksiz bağımlılıklardan bilinçli olarak kaçınır.

- Tailwind CSS

- jQuery
